Como resolver, “sendo [math]3^x + 3^{-x} = 7[/math], o valor de [math]3^{3x} - 3^{-3x} =[/math]”?
*A*
Pra fazer isso, seria possível calcular o valor de x, mas isso não é nem um pouco necessário. Basta saber o valor de [math]3^x[/math] (talvez alguém, em outra resposta, dê algum modo ainda mais simples).Vamos considerar [math]u = 3^x[/math]. Assim, [math]u^3 = (3^x)^3 = 3^{3x}[/math]. Mas depois eu volto nisso. Um pequeno detalhe: se isolarmos o [math]x[/math], aparece a relação [math]x = \log_3{u}[/math], o que significa que, para trabalharmos no conjunto dos reais, o valor de [math]u[/math] tem que ser positivo.Agora, a gente tem a equação [math]3^x + (3^x)^{-1} = 7[/math] , que se torna [math]u + u^{-1} = 7[/math]. Que pode ser reescrita como [math]u + \dfrac{1}{u} = 7[/math]. Que, depois de reorganizada, se torna a equação quadrática [math]u^2 - 7u + 1 = 0[/math]. Dela, obtemos duas soluções para [math]u[/math], ambas positivas:[math]u = \dfrac{7 - 3\sqrt{5}}{2}[/math] ou [math]u = \dfrac{7 + 3\sqrt{5}}{2}[/math]Depois de um longo processo de multiplicação, temos que [math]u^3 = 161 - 72\sqrt{5}[/math], para a primeira solução, ou, para a segunda, [math]u^3 = 161 + 72\sqrt{5}[/math].O exercício pede para encontrar o valor de [math]3^{3x} - 3^{-3x}[/math], ou também [math]3^{3x} - \dfrac{1}{3^{3x}}[/math]. Mas, como descobrimos no começo, [math]3^{3x} = u^3[/math], e já conhecemos esse(s) valor(es). Se chamarmos o valor procurado de [math]k[/math], temos que:[math]k_1 = 161 - 72\sqrt{5} - \dfrac{1}{161 - 72\sqrt{5}}[/math]ou[math]k_2 = 161 + 72\sqrt{5} - \dfrac{1}{161 + 72\sqrt{5}}[/math]Para [math]k_1[/math], após muito trabalho braçal, tem-se que:[math]k_1 = \dfrac{51840 - 23184\sqrt{5}}{161 - 72\sqrt{5}}[/math]Podemos racionalizar esse resultando, multiplicando [math]k_1[/math] por [math]\dfrac{161 + 72\sqrt{5}}{161 + 72\sqrt{5}}[/math]:[math]k_1 = \dfrac{8346240 - 3732624\sqrt{5} + 3732480\sqrt{5} - 8346240}{161^2 - (72\sqrt{5})^2}[/math][math]k_1 = \dfrac{-144\sqrt{5}}{25921 - 25920}[/math][math]k_1 = -144\sqrt{5}[/math]Pelo mesmo processo (cansativo, por sinal), encontramos [math]k_2 = 144\sqrt{5}[/math]. Portanto, o valor da expressão dada é [math]\pm 144\sqrt{5}[/math].
